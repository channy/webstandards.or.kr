웹 표준을 위하여!

한국 웹 표준 기술의 전파와 보편적 웹 접근성을 추구하는
커뮤니티 모임과 이를 위한 활동입니다.

열린 웹, 참여와 소통

웹은 누구나 참여하여 함께 만들고 나눌 수 있는 착한 기술입니다.
여러분도 함께 해 주십시오.

한국 웹 표준 프로젝트

국내 웹 표준 커뮤니티 리더들이 함께 운영하는 협의체로 다양한 공동 협력 사업을 추진합니다.

한국 Mozilla 커뮤니티

Mozilla 철학에 따른 개방형 웹 및 웹 표준에 대한 지원 등을 함께 하고 있습니다.

CSS Design Korea

CSS 기반 웹 디자인 기법에 대해 토론 및 전파, 업계 종사자의 친목을 도모합니다.

한국 웹 접근성 그룹

국내 웹 사이트의 장애인 접근성 향상을 위해 현업 개발자들이 연구 및 홍보활동을 합니다.

클리어보스

웹 표준 기술에 대해서 연구하고 자료 번역 및 공유, 개발자 스터디 등을 개최합니다.

한국 HTML5ROCKS

HTML5ROCKS 내 최신 웹 기술 자료 한국어 번역을 위한 커뮤니티입니다.

주요 활동

국내 웹 표준 기술 전파 및 스터디등을 진행하는 각종 활동 및 행사를 소개합니다.

    해결법을 제시하여, 비IE 브라우저
    지원을 위한 토론 및 연구

    Mozilla 웹표준 포럼(2002-)

  • 국내 최초의 무료 웹 표준 가이드로서
    크로스 브라우징 및 HTML, CSS, JS의
    웹 표준 기반 기법을 소개

    실전 웹표준 가이드(2003-05)

  • 국내 웹 사이트의 웹 접근성 증진을 도모하기 위해
    한국 웹 접근성 그룹(KWAG, 콱)에서 진행한
    오프라인 워크숍

    웹 접근성 워크샵 (2006-10)

  • 한국 Microsoft, Opera Software Korea,
    한국 Mozilla 커뮤니티가 개최한
    웹 기술 변화에 대한 기술 행사

    미래웹포럼 (2007-10)

  • 국내 웹 기업, 커뮤니티 및 개발자가
    함께 어울러진 기술 위주의
    본격적인 참여형 개발자 콘퍼런스

    웹앱스콘 (2007-2008)

  • 오픈 웹 기술 문서화 및 전파를 위한
    국내 개발자 자원 봉사 MDN 활동
    문서 번역 및 가이드 제작

    한국 Mozilla 개발자 네트워크 (2008-)

  • 웹 표준에 대한 지식공유와 사례공유를 목적으로 하는
    한국에서 최초로 진행된 비영리 웹 표준 세미나

    웹표준의 날(2006-10)

  • 설명

    한국 Mozilla Hacks(2009-)

  • 웹 표준과 웹 접근성의 발전을 위해 한 해동안 고생한
    동료, 친구, 선,후배가 모여서 진행하는 송년회 모임

    원앱파티(2010-)

  • 국내 최초의 무료 HTML5 가이드로서
    HTML5 기술 현황 및 CSS3
    모바일 웹 적용 기법을 소개하였습니다.

    실전 HTML5 가이드(2010)

  • 웹 퍼블리셔를 위해 HTML5 표준 명세를
    더 쉽게 이해하기 위해 만든
    한국어 웹 레퍼런스입니다.

    HTML5 오픈레퍼런스(2010-11)

  • 국내 최초 HTML5 기술에 대한 소개
    및 전망, 모바일 웹 적용 방법에
    대한 개발자 콘퍼런스

    HTML5 오픈 콘퍼런스(2010)

  • 설명문

    웹퍼블리셔 오리엔테이션 (2012-)

    새로운 오프라인 토크 모임으로,
    웹 접근성과 관련된 다양한 키워드를 중심으로
    웹 접근성 전문가와 관객들이 소통하는 모임입니다.

    다와 (Talk About Web Accessibility. 2013-)
